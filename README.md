# stem-cell-coding-agent

A test project for 2026 JetBrains Intership application, consisting of developing a stem cell - like coding agent, starting as a general code analyzer and developing into a specialised subrole. This happens by the agent analyzing a git repository and autonomously developing required specialisation tools, for example:

upon detecting SQL files -> Security Hardener
upon detecting CSS/HTML -> UI/UX Performance Optimizer
